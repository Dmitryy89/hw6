Инструкция

Чтобы произвести вычисления очистите поле ввода кнопкой "С".
Затем введите число, при этом отрицательные числа начните со знака "-" (кнопка "+/-").
Для введения дробных чисел используйте кнопку ".".
Затем нажмите на кнопку необходимого оператора и введите второе число.
После этого нажмите на кнопу "=". Произойдет вычисление.
Функция памяти (кнопка "м") работает только с операторами "+", "-", "/", "%", "*", и "an".
При использовании кнопки "←" для удаления числа из поля ввода убедитесь, что вы
полностью очистили поле.
После проведенного вычисления используйте кнопку "С" для очистки поля ввода или
продолжайте вычисления с числом в поле ввода.
